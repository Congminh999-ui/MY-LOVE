🎨 Giải thích CSS – Thuộc tính ngoài cơ bản
1. CSS Variables (Biến CSS)
css:root {
  --do:        #cc0000;
  --xanh-dam:  #1a5276;
}

:root là phần tử gốc của trang, khai báo biến ở đây để dùng được toàn trang
--tên-bien: giá_trị → đặt tên biến
Dùng lại bằng: color: var(--do);
Lợi ích: muốn đổi màu toàn site chỉ sửa 1 chỗ duy nhất


2. Linear Gradient
cssbackground: linear-gradient(135deg, var(--xanh-dam) 60%, var(--xanh-nhat));

linear-gradient → tạo nền chuyển màu
135deg → góc nghiêng (0° = xuống, 90° = sang phải, 135° = chéo góc)
60% → màu đầu chiếm 60% rồi mới bắt đầu chuyển sang màu sau


3. Object-fit
css.header-imgs img {
  width: 70px; height: 70px;
  object-fit: cover;
}

Khi ép ảnh vào kích thước cố định, ảnh hay bị méo
object-fit: cover → cắt ảnh vừa khung, không bị méo (giống background-size: cover)


4. Transition
css.sidebar ul li a {
  transition: background 0.2s;
}

Tạo hiệu ứng mượt khi thuộc tính thay đổi
background 0.2s → đổi màu nền mất 0.2 giây thay vì thay đổi đột ngột
Cú pháp: transition: thuộc-tính thời-gian;


5. Pseudo-element ::before
css.sidebar ul li a::before {
  content: "• ";
  color: var(--xanh-nhat);
}

::before → chèn nội dung trước phần tử mà không cần thêm thẻ HTML
content: "..." → bắt buộc phải có, chứa nội dung muốn chèn
Ở đây dùng để tạo dấu • màu xanh trước mỗi link sidebar


6. Overflow hidden
css.sidebar {
  border-radius: 6px;
  overflow: hidden;
}

border-radius bo góc khung sidebar
Nhưng phần tử con (.sidebar-title) có màu nền đỏ sẽ tràn ra ngoài góc bo
overflow: hidden → cắt bỏ phần tràn ra → góc bo hoạt động đúng


7. Rgba – Màu trong suốt
cssborder-right: 1px solid rgba(255,255,255,0.3);

rgba(R, G, B, A) → A là độ trong suốt từ 0 (ẩn hoàn toàn) đến 1 (đục hoàn toàn)
rgba(255,255,255,0.3) = màu trắng, trong suốt 70% → tạo đường kẻ mờ trên nền đỏ
